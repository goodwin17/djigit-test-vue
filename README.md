# Djigit test task

Vue app test project

Technologies used:

- Vue
- Vue Router
- Vuex
- Bootstrap
- SASS
- Vite


## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Preview Production Build

```sh
npm run preview
```
